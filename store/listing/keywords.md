# Levellio — ASO keyword strategy

> **This is a starting hypothesis, not a ranking guarantee.** Keyword
> performance depends on competition, install velocity, ratings, and store
> algorithms. Treat the groups below as a v1 bet to **measure and iterate**
> after launch (watch impressions → product-page views → installs per term, and
> revise each release).

## Positioning

Levellio sits at the intersection of two high-intent searches: **habit
tracker** (utility) and **RPG / gamification** (motivation). The strategy is to
rank for "habit tracker" qualified by the game angle, and to capture the
"gamified habits / level up your life" niche where competition is lighter.

## Primary terms (highest intent, anchor the listing)

- `habit tracker` — the core category term; highest volume, highest competition.
- `habits` / `routine` — broad habit-building intent.
- `gamified habits` / `gamify` — our differentiator; lower competition.
- `level up` / `RPG` — the game hook that sets us apart from plain trackers.
- `streak` — strong retention/intent signal for habit apps.

## Secondary terms (support + long tail)

- `self improvement`, `goals`, `productivity`, `focus`, `discipline`,
  `motivation`, `daily routine`, `quests`, `consistency`, `personal growth`.

## Brand

- `Levellio` — defend the brand term; expect to rank #1 quickly.

## Placement notes

### Apple App Store
- **App name (≤30):** `Levellio: Habit Quest RPG` — name terms are indexed, so
  this carries **habit**, **quest**, and **RPG** for free.
- **Subtitle (≤30):** `Turn habits into hero quests` — adds **habits** + **hero
  quests** and reads as a benefit.
- **Keywords field (≤100):** comma-separated, **no spaces**, **no plurals or
  repeats of name/subtitle words** (Apple stems and recombines automatically).
  Canonical value (generated into `app-store.md`):
  `habit tracker,routine,gamify,streak,goals,self improvement,productivity,focus,discipline,motivation`
- Don't waste the field on words already in the name/subtitle (habit, quest,
  rpg, hero) — Apple cross-combines name + keywords into search phrases.

### Google Play
- **Title (≤30):** `Levellio: Habit Tracker RPG` — Play weights the title
  heavily, so the exact phrase **habit tracker** is placed here.
- **Short description (≤80):** benefit-led and keyword-bearing (habits, quests,
  XP, streaks, hero).
- **Full description (≤4000):** Play indexes the full text, so the description
  naturally repeats core terms (habit, quest, routine, streak, level up) without
  stuffing. Keep density natural — keyword stuffing risks rejection.

## Things to test post-launch

1. Apple subtitle A/B: "Turn habits into hero quests" vs. a streak-led variant.
2. Whether `gamified habits` / `level up your life` out-converts generic
   `habit tracker` traffic (lower volume, higher intent).
3. Localized keyword sets for the top non-English markets.
